# Commands
**css_countdown**

`Usage: /countdown <SECONDS>` Start timer

**css_countdown_text**

`Usage: /countdown <SECONDS> "<TEXT>"` Start timer with text

**css_stopwatch**

`Usage: /stopwatch` Start and Stop stopwatch


| Command      | Permission   |
| ------------ | ------------ |
| `css_countdown`    | @css/root     |
| `css_countdown_text`    | @css/root     |
